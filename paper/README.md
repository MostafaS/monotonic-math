# M² Paper — Build Instructions

LaTeX source for *Monotonic Math: A Provably Floor-Preserving Tokenomic Primitive for Revenue-Bearing On-Chain Assets*.

## Build

From this directory:

```bash
cd /Users/mostafa/Documents/Personal_Projects/M2/paper
latexmk -pdf main.tex
```

This runs `pdflatex` → `bibtex` → `pdflatex` → `pdflatex` in the right order and produces `main.pdf`. Expect "undefined references" warnings on the first pass — they resolve automatically on the second `pdflatex` run.

Manual one-shot (equivalent):

```bash
pdflatex main && bibtex main && pdflatex main && pdflatex main
```

To clean intermediate files (keeps the PDF): `latexmk -c`
To delete the PDF too: `latexmk -C`

### MacTeX PATH gotcha

If you just installed MacTeX (`brew install --cask mactex-no-gui`) and see `zsh: command not found: latexmk`, the binaries are installed at `/Library/TeX/texbin` but your current shell hasn't picked up the new PATH.

**One-shot fix in the current shell:**

```bash
eval "$(/usr/libexec/path_helper)"
```

**Permanent fix:** open a fresh Terminal window (zsh re-sources login files), or add to `~/.zshrc`:

```bash
export PATH="/Library/TeX/texbin:$PATH"
```

Verify with `which latexmk` — you should see `/Library/TeX/texbin/latexmk`.

## Class file

Currently uses `article.cls` (compiles on any TeX install). For FC 2027 submission, swap to `llncs.cls` from the Springer LNCS authors page:

1. Download `llncs.cls` and `splncs04.bst` from `https://www.springer.com/gp/computer-science/lncs/conference-proceedings-guidelines`.
2. Replace the documentclass line in `main.tex`:
   ```
   \documentclass[11pt,a4paper]{article}        % current
   \documentclass[runningheads]{llncs}          % FC submission
   ```
3. Adjust bibliography style: `\bibliographystyle{splncs04}` instead of `plainnat`.
4. Remove `\maketitle`-incompatible front-matter; LNCS uses its own title block.

The section structure and macro definitions are LNCS-portable as written.
